### 시간표 생성기
# 엑셀로 만든 간단한 시간표 정리 파일입니다.

# 두 테이블이 있습니다. (시간표, dataset)

# 시간표 테이블에서는 이름만 입력합니다.
# 이름을 입력하면 dataset 테이블에서 값을 찾아 표시해줍니다.

# dataset에는 28명까지 입력할 수 있는 시간표 자료가 들어갑니다.
# 이름 이후로 수강하는 시간에 값을 넣어주면 됩니다.


### 주의사항
# dataset은 이름순 오름차순 정렬이 되어 있어야 파일이 정상적으로 작동합니다.
# 대부분의 오작동은 오름차순 정렬이 되지 않아 일어납니다.
# 그래도 잘 모르겠으면 제작자에게 문의하세요.
